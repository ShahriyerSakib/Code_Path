Basic Structure

Every HTML document follows a standard structure that defines the webpage.

Basic HTML Structure 
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

Text Elements

Basic elements for displaying and formatting text content.

Text Formatting Examples 
<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>
<br>Line break
<hr>Horizontal line

Links

Create clickable links to navigate between pages and resources.

Link Examples 
<!-- Basic link -->
<a href="https://example.com">Visit Example</a>

<!-- Link with target -->
<a href="page.html" target="_blank">Open in new tab</a>

<!-- Email link -->
<a href="mailto:hello@example.com">Email us</a>

<!-- Phone link -->
<a href="tel:+1234567890">Call us</a>

Images

Display and optimize images on your webpage.

Image Examples 
<!-- Basic image -->
<img src="photo.jpg" alt="Description">

<!-- Image with size -->
<img src="logo.png" alt="Logo" width="200" height="100">

<!-- Responsive image -->
<img src="image.jpg" alt="Photo" style="max-width: 100%">

<!-- Image as link -->
<a href="large.jpg">
    <img src="thumb.jpg" alt="Thumbnail">
</a>

Lists

Organize content with ordered and unordered lists.

List Examples 
<!-- Unordered list -->
<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

<!-- Ordered list -->
<ol>
    <li>First step</li>
    <li>Second step</li>
</ol>

<!-- Description list -->
<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language</dd>
</dl>

Tables

Display structured data in rows and columns.

Table Example 
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Sarah</td>
        <td>30</td>
        <td>London</td>
    </tr>
</table>

Forms

Collect user input with various form elements.

Contact Form 
<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <label for="message">Message:</label>
    <textarea id="message" name="message"></textarea>
    
    <button type="submit">Send</button>
</form>
